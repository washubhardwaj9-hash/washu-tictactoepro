About the developer

The classic, sharpened

Washu Tic Tac Toe Pro is an original strategy game developed by Washu Bhardwaj in 2026. This project is a completely independent game and is not affiliated with any university or educational institution.

The game keeps the three-by-three grid you grew up with, but wraps it in fast, tactile rounds designed for a phone in your hand. Every game is a short, tense back-and-forth where a single misplaced mark decides the outcome. Climb through difficulty tiers, track your win streak, and unlock new boards as proof of every victory.
